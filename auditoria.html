Auditoria

Registro de todas as acoes criticas do sistema para rastreabilidade

Visao Geral

O sistema de auditoria registra todas as acoes criticas para garantir rastreabilidade, compliance e facilitar investigacao de problemas.

Acoes Auditadas

IMPORT GERAR_LINK CONVITE_ENVIADO CONVITE_ACEITO CONVITE_RECUSADO MATCH_DESFEITO GRUPO_LOCKED GRUPO_UNLOCKED

Estrutura da Tabela

CREATE TABLE match_auditoria (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  evento_id BIGINT NOT NULL,
  pax_id BIGINT,
  grupo_id BIGINT,
  convite_id BIGINT,
  acao VARCHAR(50) NOT NULL,
  dados_json JSON,
  ip_address VARCHAR(45),
  user_agent TEXT,
  admin_id BIGINT,
  created_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,

  INDEX idx_evento_acao (evento_id, acao),
  INDEX idx_pax (pax_id),
  INDEX idx_created (created_at)
);
