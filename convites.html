Sistema de Convites

Fluxo de convite e aceite mutuo para formacao de match

Visao Geral

O sistema de convites implementa o fluxo "Quero ficar com Fulano" - onde um participante busca outro por nome e envia um convite direto. O match so e fechado com aceite mutuo.

Fluxo Principal

Buscar -> Convidar -> Aguardar -> Aceite -> Match!

Status do Convite

SENT Convite enviado, aguardando resposta
ACCEPTED Aceito - match fechado
DECLINED Recusado pelo destinatario
EXPIRED Expirou sem resposta
CANCELLED Cancelado pelo remetente

Endpoints

GET /guest/pessoas?query=nome

Busca participantes elegiveis do evento

// Response 200
{
  "pessoas": [
    {
      "paxId": 123,
      "nome": "Maria Silva",
      "statusMatch": "PRONTO"
    }
  ]
}
POST /guest/convites

Envia convite para outro participante

// Request
{ "toPaxId": 123 }

// Response 201
{
  "conviteId": 456,
  "status": "SENT",
  "expiresAt": "2026-02-10T10:00:00Z"
}

// Response 409 (Motor bloqueou)
{
  "code": "INVITE_LIMIT_REACHED",
  "message": "Voce atingiu o limite de convites pendentes"
}
POST /guest/convites/{id}/aceitar

Aceita um convite recebido

// Response 200
{
  "grupoId": 789,
  "status": "MATCH_FECHADO",
  "membros": [
    { "paxId": 100, "nome": "Joao" },
    { "paxId": 123, "nome": "Maria" }
  ]
}
POST /guest/convites/{id}/recusar

Recusa um convite recebido

// Response 200
{
  "conviteId": 456,
  "status": "DECLINED"
}
