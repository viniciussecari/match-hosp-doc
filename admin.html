Painel Admin

Console operacional para agencia/cliente gerenciar o match

Visao Geral

O painel admin permite que a agencia/cliente gerencie todo o processo de match: importar participantes, gerar links, visualizar status, travar grupos e exportar dados.

Dashboard

  • - Visao geral do evento
  • - Contadores por status
  • - Lista de participantes

Importacao

  • - Importar pax elegiveis
  • - Gerar magic links
  • - Idempotente

Export

  • - Exportar rooming list
  • - Formato CSV
  • - Dados completos

Lista eventos disponiveis para o admin
